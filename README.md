# OASIS AI Platform

World-class AI automation agency platform.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide React
- React Router v6

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/components`: Reusable UI components
- `src/pages`: Page components (Landing, Portal, etc.)
- `src/lib`: Utilities
- `src/styles`: Global styles

## Features

- Public Marketing Website (Landing, About, Services, Pricing, Case Studies, Contact)
- Client Portal (Login implemented, Dashboard pending)
